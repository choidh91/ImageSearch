# ImageSearch

프로젝트 개요

네이버 API를 이용하여 이미지를 검색할 수 있는 앱입니다.
MVVM구조로 구현했습니다.

주의사항
1. 네이버 API키가 필요합니다.

기능
1. 이미지 검색
2. 검색 히스토리 저장
3. 검색시 필터

사용 기술
1. RecyclerView
2. Room
3. Dagger Hilt
4. Retrofit
5. Glide
6. Paging3
7. Coroutine
8. ViewBinding
9. ViewModel
10. LiveData
